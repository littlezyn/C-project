# Basic Requirement
To run this code i'll recommand you to download. 
*  MinGW or gcc/g++ or clang to run the code

# Other OS discussion

# Windows

if you're in windows just go the the browser and search C/C++ compiler and download it.
* Do not forgot to add bin folder to the path of your system
```
example = 'C:\MinGW\bin' like this.
```
# Linux

if you're on linux
* Open terminal and paste the command line given below;

* Firstly update your system 
```
sudo apt update && sudo apt upgrade -y //it is for debian based system
```
* After that paste this command line to Download compiler
```
sudo apt/dnf/pacman -S install gcc g++ -y //according to your system
```
# Now You're good to go , enjoy.
